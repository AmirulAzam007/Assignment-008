## SkillSphere

SkillSphere is a modern online learning platform where users can explore courses, watch lessons, and enroll in skill-based programs like Web Development, Design, Marketing, and more.

---

## 🚀 Live Website

🔗 Live URL: https://assignment-008-ten.vercel.app/

---

## 🎯 Purpose

The purpose of SkillSphere is to provide a clean, responsive, and user-friendly online learning experience where students can easily discover and access high-quality educational content.

---

## ✨ Key Features

- 🔍 Search courses by title
- 📚 Explore multiple skill-based courses
- 👨‍🏫 Instructor showcase section
- 🔐 Authentication system using BetterAuth
- 📱 Fully responsive design
- 🎨 Modern UI with Tailwind CSS and DaisyUI
- ⚡ Optimized image handling with Next.js Image component
- ❌ Custom 404 Not Found page
- 💡 Learning tips and productivity section
